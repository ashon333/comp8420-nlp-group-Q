# 👟 SoleStride — Premium Sneaker E-Commerce Portal

> **COMP8420 - Natural Language Processing**
>
> Developed with passion by **Group Q**

SoleStride is a highly premium, modern athletic and lifestyle sneaker portal that showcases the integration of advanced AI and Natural Language Processing (NLP) within contemporary e-commerce. Built with a gorgeous orange-themed glassmorphic interface, the platform offers semantic search capabilities, extensive customer review analytics, and visual image matching.

---

## 🌟 Key Features

* **Portal Nickname Access:** Simple, secure, session-persistent access utilizing local storage (no authentication overhead).
* **AI Semantic Search:** A double-toggle search bar allowing standard text matching or advanced **AI Prompt-based Semantic Search** (e.g., "I want comfortable shoes for walking on mountains" automatically maps to corresponding Trail/Comfort sneakers).
* **Review Sentiment & Authenticity Analytics:** Clicking any review opens a dedicated **AI Analysis panel** displaying:
  * **Sentiment Analysis:** Categorized as Positive, Negative, or Neutral.
  * **Recommendation Status:** Deduces whether the author recommends the product.
  * **AI Writing Detection:** Flagging reviews likely generated by LLMs.
  * **Authenticity Flagging:** Highlighting potentially fake or spam reviews.
  * **Product Relevance Matching:** Deduces if the written content actually relates to the product (e.g. typing a review about "cars" under a shoe triggers an *Irrelevant Content* warning).
* **Dynamic Reviewing System:** Add a review, submit it, and watch the AI pipeline analyze it in real time before appending it to the catalog!
* **Visual Search (Image Matching):** Upload a sneaker photo to discover similar fits inside the active database using simulated vector embedding search.
* **Filterable Catalog:** Filter sneakers by Category, Max Price slider, or Min Rating, sorted seamlessly by price or quality metrics.

## ⚙️ Local Setup and Installation

### 1. Prerequisites
* [Node.js](https://nodejs.org/) (v18 or higher)
* [Python](https://www.python.org/) (v3.10 or higher)
* Local MongoDB instance (optional)

### 2. Backend Setup
1. Navigate to the backend directory:
   ```bash
   cd ashon/backend
   ```
2. Create and activate a Python virtual environment:
   ```bash
   python -m venv venv
   # On Windows:
   .\venv\Scripts\activate
   # On macOS/Linux:
   source venv/bin/activate
   ```
3. Install the dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Create a `.env` configuration file:
   ```env
   PORT=8000
   MONGO_URL=mongodb://localhost:27017
   ```
5. Start the FastAPI server:
   ```bash
   python main.py
   ```
   *The Swagger interactive documentation will be hosted at `http://localhost:8000/docs`.*

### 3. Frontend Setup
1. Navigate to the frontend directory:
   ```bash
   cd ../frontend
   ```
2. Install the node packages:
   ```bash
   npm install
   ```
3. Create a `.env` configuration file:
   ```env
   VITE_API_URL=http://localhost:8000/api
   ```
4. Start the Vite development server:
   ```bash
   npm run dev
   ```
   *The application will boot at `http://localhost:5173`.*

## 🚀 Deployment to Vercel (Backend)

The backend includes a pre-configured `vercel.json` enabling immediate serverless deployment to Vercel:

1. Link your repository on Vercel.
2. In the **Configure Project** stage, set the **Root Directory** to `ashon/backend`.
3. Provide your environment variables (like a live cloud `MONGO_URL` hosted on MongoDB Atlas).
4. Click **Deploy**!
